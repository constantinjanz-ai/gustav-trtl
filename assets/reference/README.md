# Reference & scrapbook photos

Drop real photos here and they appear automatically — no code changes needed.

## Scrapbook (Erinnerungen) photos

Name each file after the memory it belongs to (any of `.jpg .jpeg .png .webp`).
The scrapbook shows the photo instead of the emoji as soon as the file exists:

| File name (any image extension) | Memory card |
|---|---|
| `erste-ernte`        | Die erste Ernte 🍓 |
| `erste-teezeit`      | Die erste Teezeit 🍵 |
| `lieblingsstein`     | Der Lieblingsstein 🪨 |
| `sonnenrampe`        | Tatas Sonnenrampe ☀️ |
| `loewenzahn`         | Löwenzahn-Festmahl 🌼 |
| `constantin-foto`    | Das erste Foto 📸 (a real Gustav portrait is perfect here) |
| `ausreisser`         | Der kleine Ausreißer 🚪 |
| `herbst-blaetter`    | Buntes Herbstlaub 🍂 |
| `winterkiste`        | Die Winterkiste 📦 |
| `winterschlaf`       | Gute Nacht, Gustav ❄️ |

Example: save a photo as `assets/reference/constantin-foto.jpg` → it shows on
that card next time the page loads. Square-ish crops look best (cards crop to
fill). Photos display smoothly (not pixelated).
